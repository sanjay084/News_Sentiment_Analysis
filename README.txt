Project OverviewThis project is a web-based application designed to extract, summarize, and analyze news articles related to a given company. It provides sentiment analysis, comparative insights, and text-to-speech (TTS) output in Hindi. Users can input a company name and receive a structured sentiment report with audio output.

Features
News Extraction: Retrieves the latest articles using the NewsAPI.
Sentiment Analysis: Classifies articles as Positive, Negative, or Neutral.
Comparative Analysis: Provides insights on sentiment distribution across articles.
Text-to-Speech: Converts summarized content into Hindi audio.
User Interface: Simple and interactive interface using Streamlit.

Technologies Used
Programming Language: Python
Web Framework: Streamlit
APIs: NewsAPI for news extraction
Libraries:
newsapi-python for fetching news
textblob for sentiment analysis
gTTS for Hindi TTS
pandas for data manipulation
